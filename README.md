## Password Strength Checker

## Decription
A simple Python program that checks the strength of a password.

## Features

- Checks password length
- Checks uppercase and lowercase letters
- Checks numbers
- Checks special characters
- Provides strength rating and suggestions

## How to Run

```bash
python Password_strength_checker.py
```

## Sample Output

```text
Enter a password: Raji@1234

Password Strength: Very Strong
Excellent! Your password meets all security requirements.
```
